# About census map
**Icon color reader**

*The following colors represent the range of a given country  population*
    
    blue <= 100
    purple <= 1000
    orange <= 10000
    darkgreen <= 100000
    beige <= 1000000
    black <= 10000000
    darkblue <= 100000000
    red <= the rest
